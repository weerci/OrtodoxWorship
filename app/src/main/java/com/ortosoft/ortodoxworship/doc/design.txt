2015.12.16 Добавлен псалтырь. При свайпе слева направо, должен изменяться язык с цекровнославянского на русский, 
при этом должен отображаться тот стих, который в настоящий момент читается.
Поэтому ListView должен быть синхронизирован на уровне стиха, для этого 
	1. Создается таблица psalms (id_psalm, id_vers, body, caption, comment) primary key(id_psalm, id_vers)
	2. Таблица-связка psalms_worships(id_worship, id_psalm, by_ord) 
	3. Изменяется таблица worships, добавляется столбец id_worship, для возможности создания иерерхии псалом/кафизма
